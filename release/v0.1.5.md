## loglm v0.1.5

This is a small bugfix release for Claude Code users.

### Fixes

- Fixed Claude Code session resume detection when the project path contains spaces, tildes, or other non-safe characters.
- Restores automatic `claude --continue` behavior when Claude Code history already exists for the current directory.

### Validation

- Added regression coverage for Claude Code history detection with spaces and tildes in the project path.

### Install / Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/uninstall.sh | bash
```
