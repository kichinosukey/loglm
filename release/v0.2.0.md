## loglm v0.2.0

This release adds experimental support for OpenClaw and Hermes Agent, including
logged launch, prompt-agent installation, and decoded log cleanup.

### Highlights

- Added experimental OpenClaw support
  - selectable as `openclaw`
  - launches `openclaw tui --local`
  - `--new` starts a distinct `loglm-...` OpenClaw session key
  - prompt-agent install can register an OpenClaw skill through
    `openclaw skills install`
- Added experimental Hermes Agent support
  - selectable as `hermes`
  - launches `hermes --continue` only when previous Hermes session history is found
  - otherwise launches `hermes` normally, avoiding first-run failure when no
    previous CLI session exists
  - prompt-agent install writes a Hermes skill wrapper under
    `~/.hermes/skills/research/<repo-name>/SKILL.md`
- Extended prompt-agent source resolution
  - `AGENT_INSTALL_OPENCLAW.md` for OpenClaw-specific instructions
  - `AGENT_INSTALL_HERMES.md` for Hermes-specific instructions
  - both fall back to `AGENT_INSTALL.md`
- Improved `loglm-decode`
  - adds OpenClaw log cleanup
  - adds Hermes Agent log cleanup
  - removes more TUI status/spinner/banner noise while keeping prompts,
    tool-use traces, and responses readable

### Platform Status

- Existing stable targets remain:
  - macOS
  - Ubuntu
  - Ubuntu on Lima (macOS)
  - Ubuntu on WSL2
  - Raspberry Pi OS
  - Chrome OS Linux container / Crostini
- OpenClaw and Hermes Agent support is experimental.
  - Tested during development on Ubuntu VM and macOS VM.
  - Upstream OpenClaw/Hermes installer behavior may still vary by platform.

### Notes

- In `loglm`, `hermes` means Hermes Agent CLI/TUI, not the Hermes model family.
- OpenClaw/Hermes prompt-agent support currently favors skill-based integration
  where the upstream agent supports it.
- `--dangerous` is not mapped to OpenClaw or Hermes Agent yet.

### Validation

- Added regression coverage for OpenClaw and Hermes launch commands.
- Added regression coverage for OpenClaw and Hermes prompt-agent skill install.
- Added regression coverage for OpenClaw and Hermes decoded log cleanup.
- Existing Codex, Claude Code, Gemini, PII redaction, and timeline regression
  checks pass.
