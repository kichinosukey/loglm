## loglm v0.1.7

This is a bugfix release focused on Claude Code log readability.

### Fixes

- Fixed Claude Code prompt redraw decoding where the first character of a user prompt could be lost.
  - Example: `loglm って知ってる？` could appear as `oglmって知ってる？`.
- Fixed overlap trimming for Claude Code logs so tail-only matches do not discard the current session body.
  - This prevents decoded logs from becoming almost empty when the raw log contains substantial content.

### Improvements

- Repository `AGENTS.md` now includes loglm runtime notes, matching the managed notes generated by normal `loglm` launch.

### Validation

- Added regression coverage for Claude Code prompt redraw decoding.
- Added regression coverage for Claude Code tail-only overlap trimming.

### Install / Uninstall

```bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/ks91/loglm/main/uninstall.sh | bash
```
